\section{Write together}
\label{sec:write-together}

Welcome to \LocalLeaf. This starter file is a small, compile-safe tour of the LaTeX commands that LocalLeaf helps you insert and autocomplete. Edit this paragraph, press Ctrl+S, and recompile to see the PDF update.

\subsection{Formatting examples}
Use \textbf{bold}, \textit{italic}, \emph{emphasis}, \underline{underline}, and \texttt{monospace} text when you need different styles. You can also add a link with \href{https://www.latex-project.org/}{the LaTeX Project} or write a plain URL with \url{https://www.latex-project.org/}. Footnotes work too.\footnote{This is a sample footnote.}

\subsubsection{References and citations}
This section is labeled as \texttt{sec:write-together}. You can reference it with Section~\ref{sec:write-together}, page~\pageref{sec:write-together}, or \autoref{sec:write-together}. Equations can be referenced with \eqref{eq:sample-model}. Citations work with \cite{localleaf2026}, \citep{knuth1984}, \citet{localleaf2026}, \parencite{knuth1984}, and \textcite{localleaf2026}.

\paragraph{Lists}
Use itemized lists for quick notes:
\begin{itemize}
  \item Open a project.
  \item Host a session.
  \item Invite collaborators.
\end{itemize}

Use numbered lists for ordered steps:

  \item Save the file.
  \item Recompile the PDF.
\end{enumerate}

\section{Math examples}
Inline math uses \( \alpha + \beta = \gamma \). Display math can use fractions, roots, sums, integrals, Greek letters, comparison symbols, multiplication, infinity, and auto-sized delimiters:
\begin{equation}
  f(\theta) = \left( \frac{\sqrt{x^2 + y^2}}{1 + \lambda} \right)
  + \sum_{i=1}^{n} i
  + \int_{0}^{\infty} e^{-t}\,dt,
  \qquad \mu \neq \pi,\ \sigma \leq \omega,\ \delta \geq \epsilon,\ x \times y
  \label{eq:sample-model}
\end{equation}
