# Provider logo notices

The provider marks in this directory were copied from the `ln-dev7/logos-apps` catalog at commit `fd7b5aa517a25ce94f2af22bb00d63d9aae201ed`:

- https://logos.lndev.me/
- https://github.com/ln-dev7/logos-apps

The catalog code and organization are MIT licensed, but the logos themselves are not covered by that license. Each logo remains the trademark and property of its respective owner. LocalLeaf uses these marks only to identify a provider option and does not imply sponsorship, affiliation, or endorsement.

Do not recolor, distort, combine with the LocalLeaf mark, or use these assets outside the provider-identification context without checking the current owner guidelines.

## Per-owner review

This table records the owner material reviewed for each mark; it is an audit trail, not legal advice or a substitute for permission. “Identification only” means the mark is shown beside the provider's own name in a provider picker, at lower prominence than the LocalLeaf brand, and is never used as a LocalLeaf endorsement or product mark.

| Asset | Owner reference reviewed | Redistribution/use conclusion |
| --- | --- | --- |
| `openai.svg` | [OpenAI design guidelines](https://openai.com/brand/) | Guidelines permit direct reference to OpenAI services subject to the marks terms. Keep the artwork unchanged and use it only to identify the OpenAI provider. |
| `lm-studio.svg` | [LM Studio brand guidelines](https://lmstudio.ai/brand) | The page grants conditional use of the marks and specifically permits reference to LM Studio. Keep the artwork unchanged, legible, and clearly separate from LocalLeaf branding. |
| `cursor.svg` | [Cursor brand guidelines](https://cursor.com/en-US/brand) | Cursor publishes assets for consistent representation. Use the mark unchanged and only to identify the Cursor provider. |
| `opencode.svg` | [OpenCode brand assets](https://opencode.ai/brand) | OpenCode publishes downloadable PNG/SVG brand assets. Use the mark unchanged and only to identify the OpenCode provider. |

OpenRouter and Ollama use LocalLeaf's neutral generic provider glyph because the reviewed owner material did not provide an express third-party redistribution grant for their marks.

Review date: 2026-07-11. Recheck the linked owner terms before each public release because brand rules can change independently of this repository.
